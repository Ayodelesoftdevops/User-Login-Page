🔐 Login Page Project

This is a simple login page built with HTML, CSS, and JavaScript. It includes basic form validation and a clean UI design with a background image and navigation menu.

🚀 What this project does
Displays a login form with username and password fields
Shows real-time validation errors for incorrect input
Checks:
Username is not empty and must be at least 3 characters
Password must contain letters, numbers, and be at least 6 characters
Shows a success message when login is valid
Redirects user to a home page after successful login


🛠️ Technologies used
HTML5
CSS3
JavaScript (Vanilla JS)

🎯 How it works
User enters username and password
JavaScript validates input on form submission
Errors are shown below each field if input is invalid
If everything is correct:
A success alert appears
User is redirected to home.html

⚡ Features
Simple and clean UI
Form validation without page reload
Error messages under input fields
Responsive layout (basic level)
Background image with centered login card

📌 Notes
This is a front-end only project (no backend yet)
You can later connect it to Node.js + MongoDB for real authentication
Password validation uses a regex pattern for basic security rules
